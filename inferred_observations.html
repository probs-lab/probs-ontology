Inferred observationsÂ¶

The Observations already described can represent the relevant information about a Material flow systems in the form in which it is published â but often the way we want to query it is different. For example, detailed data is published on production of many types of pharmaceuticals, but if this is too much irrelevant detail for a given study, we might prefer to query for simply the total production of all types of pharmaceuticals.

To achieve this, the Inferring new observations algorithm can infer new Observations from the starting ones, based on the structure of the data (Composition and equivalence).

To distinguish between the original data and the inferred facts, datasources are entered as DirectObservations, while inferred facts are created as InferredObservation.
